# WatchPost

Lightweight cross-platform desktop monitor for your PC and remote servers — `btop`-style stats plus a desktop widget.

Built with [Tauri](https://tauri.app/) (Rust + system webview).

## Development

```bash
export PATH="/opt/homebrew/bin:$PATH"
npm install
npm run tauri dev
```

## Tests

```bash
npm test
```

## Build

```bash
npm run tauri build
```
